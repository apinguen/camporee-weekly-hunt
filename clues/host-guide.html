How It Works

Each riddle page does a live fetch() check for the next file when a player solves it. If the file exists ‚Üí the link appears. If not ‚Üí a "return next week" message appears instead. You control timing simply by uploading each file on the right week.

One-Time Setup: Add Your Google Doc Link

Open none.html and find this line near the bottom:

const PRIZE_URL = "YOUR_GOOGLE_DOC_URL_HERE";

Replace YOUR_GOOGLE_DOC_URL_HERE with your actual Google Doc share link (the one with Anyone with the link can view enabled).
